VRF-ed25519

Verifiable Random Functions (VRF) allow key-pair owners, (skVRF​,vkVRF​), to evaluate a pseudorandom function in a provable way given a randomness seed. Any party with access to the verification key, vkVRF​, the randomness seed, the proof and the generated randomness can indeed verify that the value is computed as expected. The VRF specification has changed overtime in Cardano, and the nodes use a different algorith, pre-Babbage and post-Babbage (if there exists a reference to the HF, we should point it out). We expose both specifications, and the motivations for such a change.
